# ECS-Microservice
## 概要
### 内容 
- ECSのマイクロサービスアーキテクチャを想定した実装を実施する。
- 方法の一つとして、**Service Connect**を使用する。
- Goの実装は考えてあるので、簡単な実装を実施次第ECS構築をまずは手動で
- その後Terraformで構築
